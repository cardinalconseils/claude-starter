The CKS wiki is a living knowledge base for this project. Pages are lazy-initialized and populated by agents during sprint learnings, design reviews, and retrospectives. Use `/cks:wiki` commands to read, write, list, and search pages. The wiki index is at `memory/wiki/index.md`.
