# DESIGN.md Examples — Condensed Style Summaries

Reference examples from real design systems. Use these to understand the range of approaches and to inspire generation when the user says "like [brand]."

## Claude (Anthropic)
- **Vibe**: "Literary salon reimagined as product page — warm, unhurried, intellectual"
- **Mode**: Light-first, parchment canvas (#f5f4ed)
- **Signature**: Custom serif (Anthropic Serif 500), terracotta brand accent (#c96442), ring-based shadows
- **Colors**: Warm neutrals (no cool blues except focus), olive gray text (#5e5d59), ivory surfaces (#faf9f5)
- **Typography**: Serif/sans hierarchy, relaxed 1.60 body line-height for editorial feel
- **Components**: Asymmetric button padding, generous border-radius (8-32px), whisper-soft shadows
- **Don'ts**: No cool grays, no weight 700+ on serifs, no sharp corners

## Linear
- **Vibe**: "Dark-mode-native tool for engineers — minimal, precise, fast"
- **Mode**: Dark-first, marketing black (#08090a)
- **Signature**: Inter Variable with OpenType cv01/ss03, sparse indigo accent (#5e6ad2), signature weight 510
- **Colors**: Luminance stacking (darkness as space), quaternary text hierarchy
- **Typography**: 24 styles from Display XL (72px) to Tiny (10px), Berkeley Mono for code
- **Components**: Ghost/subtle/primary button variants, luminance-based elevation (not shadows)
- **Don'ts**: No traditional box shadows, no color overuse, no weight 700 body text

## Stripe
- **Vibe**: "Premium fintech — clean, trustworthy, developer-friendly"
- **Mode**: Light-first, white canvas
- **Signature**: Gradient brand accents (blue-purple-cyan), blurple primary (#635bff), fine borders
- **Colors**: Cool neutral grays, accessibility-first contrast, prominent status colors
- **Typography**: System font stack (-apple-system), tight heading line-heights, generous body
- **Components**: Subtle rounded corners (6px), hairline borders, soft shadows with blue tint
- **Don'ts**: No gradients in functional UI (only marketing), no heavy shadows, no warm tones

## Vercel
- **Vibe**: "Infrastructure minimalism — black/white with surgical precision"
- **Mode**: Dark-first, true black (#000) background
- **Signature**: Geist font family (Sans + Mono), blue accent (#0070f3), sharp 6px radius
- **Colors**: True black/white, minimal mid-grays, single blue accent for all interactive
- **Typography**: Geist Sans for UI, Geist Mono for code/data, tight letter-spacing
- **Components**: 1px borders, subtle hover states, minimal shadow usage
- **Don'ts**: No color variety (blue only), no rounded pills, no decorative elements

## Notion
- **Vibe**: "Blank canvas productivity — quiet, adaptable, content-first"
- **Mode**: Light-first, warm white (#ffffff with warm tint)
- **Signature**: Content-first with minimal chrome, emoji-as-icon system, warm gray palette
- **Colors**: Warm grays (#37352f text), pastel accent palette for tags/labels, subtle borders
- **Typography**: System fonts, 16px body, comfortable 1.5 line-height, minimal weight variation
- **Components**: Very subtle borders, hover-reveal interactions, generous padding
- **Don'ts**: No heavy borders, no prominent shadows, no competing visual elements

## Supabase
- **Vibe**: "Open-source Firebase — developer-friendly, dark-mode, vibrant"
- **Mode**: Dark-first, deep gray (#1c1c1c)
- **Signature**: Brand green (#3ecf8e), code-first documentation style, gradient accents
- **Colors**: Dark surfaces with green/emerald accents, syntax-highlighting-inspired status colors
- **Typography**: Custom font + monospace heavy, generous code block styling
- **Components**: Rounded corners (8px), green CTAs, dark card surfaces, glow effects
- **Don'ts**: No light mode as afterthought, no small code text, no muted brand colors

## Raycast
- **Vibe**: "Power-user launcher — fast, dark, keyboard-first"
- **Mode**: Dark-first, deep black with vibrancy
- **Signature**: Vibrant gradient accents, frosted glass surfaces, tight compact UI
- **Colors**: True black base, vibrant multi-color accents, frosted overlays
- **Typography**: SF Pro / system fonts, compact sizing, tight line-heights for density
- **Components**: Glassmorphism panels, compact list items, keyboard shortcut badges
- **Don'ts**: No spacious layouts, no light backgrounds in primary UI, no heavy fonts

## Figma
- **Vibe**: "Creative collaboration — approachable, colorful, playful-professional"
- **Mode**: Light-first, neutral white
- **Signature**: Multi-brand purple (#a259ff), playful illustrations, tool-like precision
- **Colors**: Neutral canvas with vibrant purple accent, status colors for multiplayer
- **Typography**: Inter for UI, clear hierarchy, comfortable sizing
- **Components**: Rounded corners (8px), clear interactive states, floating panels
- **Don'ts**: No dark UI in editor chrome, no monospace outside code, no heavy shadows
