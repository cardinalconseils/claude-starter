## What Changed
n8n's 2026-05-05 release introduced the Microsoft Agent 365 Trigger node, which allows n8n agents to appear as members of a team inside Microsoft 365 apps (Teams, Outlook, etc.), enabling human-in-the-loop workflows triggered directly from M365.

## Impact on Agents
When designing n8n workflows for users in Microsoft 365 environments, the Microsoft Agent 365 Trigger is now available as a first-class integration point. This enables Copilot-style agent surfaces without requiring custom M365 app development.
