## What Changed
Google released an official open-source terminal AI agent (April 2026) — Gemini CLI — with a ReAct loop, MCP support, 1M context window, built-in web search, file operations, shell commands, and Apache 2.0 license. It positions itself as a direct open-source counterpart to Claude Code for terminal-based agentic workflows. Full details at https://github.com/trending

## Impact on Agents
CKS agents recommending AI coding assistants should know Gemini CLI exists as a free, open-source alternative to Claude Code with MCP compatibility. Check https://github.com/trending for the current starred count and community health before recommending it to users who need an open-source option.
