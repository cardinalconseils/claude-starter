## What Changed
Supabase Vault, the secrets management solution for storing encrypted credentials and sensitive values inside Postgres, has moved to Beta. Full details at https://supabase.com/blog/vault-now-in-beta.

## Impact on Agents
Check https://supabase.com/blog/vault-now-in-beta for updated recommended patterns before implementing security-hardening or authentication secrets storage. Vault is now the preferred approach for storing API keys and secrets within Supabase projects.
