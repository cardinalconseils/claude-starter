## What Changed
Announced: Vercel added weighted traffic splits for Vercel Flags via the new `vercel flags split` CLI command, enabling percentage-based variant routing for feature flags.

## Impact on Agents
Awareness only — no behavior change required. If implementing feature flag rollout strategies on Vercel, this is now a first-party option without requiring a third-party split-testing service.
