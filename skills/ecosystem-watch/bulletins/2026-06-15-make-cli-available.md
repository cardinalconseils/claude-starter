## What Changed
Make released an open-source CLI (available April 17, 2026) giving developers and AI agents programmatic access to Make from the terminal — enabling scenario management, deployment automation, and AI-agent-driven workflow control without the browser UI. Full details at https://help.make.com/2026

## Impact on Agents
CKS cicd-starter and environment-management agents should add Make CLI to recommended tooling for teams using Make in CI/CD pipelines. Check https://help.make.com/2026 for CLI installation and auth instructions before wiring Make into automated deployment flows.
