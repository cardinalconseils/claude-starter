## What Changed
Announced: Grok Build now recommends and uses the Vercel plugin when prompts mention Vercel, allowing Grok to draw on Vercel-specific knowledge and tooling while building apps. Full details at https://vercel.com/changelog

## Impact on Agents
Awareness only for most CKS projects. Teams using Grok Build as an alternative AI coding assistant can now leverage Vercel-native tooling within Grok sessions. Check https://vercel.com/changelog for updated recommended patterns when combining Grok Build with Vercel deployment workflows.
